# FarmaEstoque

FarmaEstoque e um projeto Java para controle de estoque farmaceutico, com foco em cadastro, consulta e organizacao de medicamentos.

## Objetivo inicial

O objetivo nesta fase e adaptar um CRUD simples de cadastro para um sistema voltado ao gerenciamento de estoque de farmacia, mantendo uma base Swing com persistencia em banco relacional.

## Tecnologias

- Java
- Swing
- MySQL
- JPA/Hibernate
- Maven

## Funcionalidades implementadas

- Login e cadastro de usuarios.
- Cadastro, busca, listagem e exclusao de produtos.
- Cadastro, busca, listagem e exclusao de fornecedores.
- Controle basico de estoque por quantidade de produto.

## Execucao parcial

1. Configure um banco MySQL local.
2. Ajuste as credenciais em `src/main/resources/hibernate.cfg.xml`.
3. Compile o projeto com Maven.
4. Execute a classe principal `br.com.farmaestoque.FarmaEstoque`.

## Origem

O projeto evoluiu a partir de um CRUD em Java, MySQL e JPA criado no NetBeans, inicialmente estruturado para cadastro de clientes.
